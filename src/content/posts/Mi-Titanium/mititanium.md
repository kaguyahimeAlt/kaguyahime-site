This page contains information for the **Xiaomi Titanium device**, which uses the Qualcomm Snapdragon MSM8953 SoC. This project aims to bring the latest software to older devices.

## Supported devices
* Redmi 5 Plus (Vince)
* Redmi S2/Y2 (YSL)
* Mi Max 2 (Oxygen)
